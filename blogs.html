Question-1: What are the different ways to select an element in the DOM?

Here are the common ways to select elements in the DOM:

  • document.getElementById();
  • document.getElementsByClassName();
  • document.getElementsByTagName();
  • document.querySelector();
  • document.querySelectorAll();

Each method has its use case depending on the structure of DOM and the specificity of the element we want to target.

Question-2: What is the difference between innerHTML, innerText and textContent?

Here's a concise breakdown of the differences between innerHTML, innerText and textContent:

  • innerHTML:
    1. Retrieves or sets the HTML content of an element, including tags.
    2. Parses and renders HTML, which can include child elements.
  • innerText:
    1. Retrieves or sets the visible text of an element, excluding hidden elements (e.g., those styled with display: none or visibility: hidden).
    2. Takes into account CSS styles like text-transform .
  • textContent:
    1. Retrieves or sets the raw text content of an element, including hidden elements.
    2. Does not parse HTML or consider CSS styles.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events on its child elements. Instead of adding event listeners to multiple child elements, the parent element listens for events and determines which child triggered the event using event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event triggered on a child element propagates (or bubbles up) through its ancestors, starting from the target element and moving up to the root of the DOM tree.

Question-5: How do you create, add and remove elements using JavaScript?

Here are some the common ways to create, add and remove elements using Javascript:

  • To create an element:
    1. - document.createElement('div');
  • To add an element:
    1. - parent.appendChild(child);
    2. - parent.append(child, 'text');
  • To remove an element:
    1. - parent.removeChild(child);
    2. - element.remove();
